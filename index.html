Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git workflow to add, commit, and push your work to your feature branch: 1) make sure you're on the appropriate feature branch in command line; 2) git add -A 3) git commit -m "example added CSS to project" 4) git pull origin main 5) git push origin feature/add-css (example) 6) navigate to respository on GitHub and click "pull requests -> "New pull request" 7) make sure base branch is 'main' and comparison branch is 'feature...'. Add title and description 8) click "create pull request" -> "merge pull request" 9) click issues tab and close issue 10) to update local main branch to to command line and: 11) git checkout main 12) git pull origin main

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes a code in a script.
